The official web platform for the **Innovation and Entrepreneurship Development Centre (IEDC)** at the **Central University of Kerala**. This project features a modern, responsive public-facing website and a fully secure backend Admin Dashboard to manage university events, announcements, and team rosters.

🌐 **Live Demo:** https://iedc-cuk.web.app/

---

## ✨ Features

### 🏢 Public Facing Site
* **Modern UI/UX:** Sleek glassmorphism design, gradient text, and a custom mouse-follower spotlight effect.
* **Smooth Navigation:** Custom smooth-scrolling architecture for seamless one-page navigation.
* **Dynamic Galleries:** View upcoming announcements, active events, and a rich "Past Events" image gallery.
* **Fully Responsive:** Optimized for both mobile devices and large desktop displays using Tailwind CSS.

### 🔐 Admin Dashboard (Protected)
* **Secure Authentication:** Firebase-powered login protecting the `/admin` routes.
* **Manage Events & Announcements:** Full CRUD operations to keep the student body updated.
* **Team Roster Management:** Invite new leads to the team, track profile setup status, and secure "Revoke Access" kill-switches.
* **Intuitive Grid Interface:** Beautiful 2x2 modular dashboard for quick admin navigation.

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (with custom utility classes and animations)
* **Icons & Motion:** Lucide-React, Framer Motion
* **Backend & Database:** Firebase (Authentication, Firestore Database)
* **Hosting & Deployment:** Firebase Hosting
